CO₂ SPAR RECHNER - BACKEND


This is the backend system of the youngcaritas Kleidertasusch CO₂ savings calculator developed as an End-of-Semester Final Project.

It is designed for administrators to manage the database of items and their corresponding base CO₂ values.

Any changes made here are supposed to directly reflect the items in the live calculator interface.

Admins can:

  • Add new items with base CO₂ values.
  • Edit or delete existing items
  • View the current CO₂ item list via a UI or SwaggerDocs
  • Approve or reject pending user registrations

Start Exploring:

  • /docs: Explore and test the API using Swagger UI

  • /UI/admin: Access the admin user interface
